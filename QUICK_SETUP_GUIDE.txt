# Quick Setup Guide: Settings & Earnings Features

## What's New?

Your RetroGameHub marketplace now has two powerful seller tools:

1. **Seller Settings Page** - Manage store info, security, and notifications
2. **Earnings & Reports Dashboard** - View sales analytics with interactive charts

## Installation (5 minutes)

### Step 1: Update Database

Go to this URL in your browser:
```
http://localhost/retroonline/migrate-add-seller-settings.php
```

You should see a success message.

### Step 2: That's it! 

The pages are ready to use.

## How to Use

### Access Settings Page

**As a Seller:**
1. Login to RetroGameHub
2. Click "Settings" in the navigation menu
3. Update your store information, password, and notifications
4. Click "Save Changes" for each section

### Access Earnings Page

**As a Seller:**
1. Login to RetroGameHub  
2. Click "Earnings" in the navigation menu
3. Select a time period: Last 30 Days, Last 12 Weeks, or Last 12 Months
4. View your sales charts, top products, and earnings forecast
5. Use the earnings breakdown table for detailed financial info

## Troubleshooting

### I see "table doesn't exist" error
**Solution:** You need to run the migration. Visit:
```
http://localhost/retroonline/migrate-add-seller-settings.php
```

### Settings form isn't saving
**Ensure:**
- Store name is not empty
- Password is at least 8 characters if changing password
- You're logged in as a seller
- JavaScript is enabled in your browser

### Charts not showing
**Ensure:**
- You have at least one order on record
- JavaScript is enabled
- You can see data in the earnings summary cards
- Try refreshing the page

### Forecast section not showing
**This is normal if:**
- You have no sales history yet
- You just started selling
- Your sales are still being processed

Once you have sales data, the forecast will calculate automatically.
